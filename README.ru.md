KozakScript — это экспериментальный язык программирования, поддерживающий несколько диалектов, вдохновлённый C++, Python и Pascal.

Изначально проект задумывался как забавная и познавательная идея: язык программирования, позволяющий людям изучать и писать код, используя синтаксис и терминологию, более близкие к их родному языку. Со временем проект развился из простого прототипа в функциональный интерпретируемый язык с поддержкой сборки исполняемых файлов, несколькими синтаксическими диалектами и генерацией автономных приложений.

В настоящее время KozakScript поддерживает:
- Украинский синтаксис
- Украинский кириллический синтаксис
- Русский синтаксис
- Русский кириллический синтаксис
- Английский синтаксис
- Символический синтаксис

Язык находится в стадии активной разработки, и постоянно добавляются новые функции, улучшения синтаксиса и модули.

# 🚀 Установка

Загрузите последнюю версию и распакуйте файлы в любое место на вашем компьютере.

Для запуска программ KozakScript или встроенных исполняемых файлов отдельная установка Python не требуется.

---

# ▶️ Running Programs

Use the interpreter executable to run `.kozak` files:

```bash
KozakScriptInterpreter.exe path\to\program.kozak
```

Example:

```bash
KozakScriptInterpreter.exe hello.kozak
```

# 📦 Упаковка исполняемых файлов

KozakScript включает встроенный упаковщик, способный упаковывать программы с расширением `.kozak` в автономные исполняемые файлы Windows.

Упакованные приложения:
- включают необходимую среду выполнения внутри себя
- не требуют установки Python
- могут включать дополнительные ресурсы и модули

## Основное использование

```bash
kozak_bundler.exe my_program.kozak
```

---

## 📁 Добавление дополнительных файлов

Вы можете встраивать внешние файлы в итоговый исполняемый файл.

### Форматирование

```bash
kozak_bundler.exe my_program.kozak --add-data source_path;destination_name
```

### Пример

```bash
kozak_bundler.exe game.kozak --add-data config.json;config.json
```

Можно добавить несколько файлов:

```bash
kozak_bundler.exe game.kozak ^
--add-data config.json;config.json ^
--add-data assets/sprite.png;sprite.png ^
--add-data sounds/hit.wav;hit.wav
```
